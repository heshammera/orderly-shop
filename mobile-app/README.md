This is a React Native (Expo) application that wraps the Social Commerce Hub store in a WebView.

## Prerequisites
-   Node.js (installed)
-   Expo Go app on your phone (iOS/Android)

## Setup
1.  Install dependencies:
    ```bash
    npm install
    ```

## Running
1.  Start the development server:
    ```bash
    npx expo start
    ```
2.  Scan the QR code with the Expo Go app.

## Configuration
-   **Store URL**: Edit `App.js` to change the `STORE_URL` constant.
-   **App Info**: Edit `app.json` to change the app name, icon, and splash screen.

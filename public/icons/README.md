To generate PWA icons, you can:
1. Use an online tool like https://realfavicongenerator.net/
2. Or use the command line tool: npx pwa-asset-generator

For now, you need to create the following icon sizes in the /public/icons/ directory:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

All icons should use the brand color (#2563eb) and feature a shopping/commerce theme.
